React.createElement() - Creates an object representation of a DOM node. Has three arguments - 1st: TAG - HTML tag that you want to create, 2nd: Any attributes that you want to add on the DOM object representation that's being created - ie. id, class and the likes, 3rd: Children of the element.

Example:

Regular HTML: 

        
          < h1 id="cris"> Cris < / h1 >

React Version of above HTML code: 
        
          const headerElement = React.createElement{ 
          'h1', 
          {id: 'cris'}, 
          'Cris'  
          }
